# SciTech Portfolio

A comprehensive digital portfolio showcasing academic achievements, STEM projects, and extracurricular involvement throughout the SciTech program (Grades 9-12) at Port Credit Secondary School.

## 🎓 Overview

This portfolio serves as permanent evidence of learning, growth, and accomplishments in the SciTech program. It documents academic progress, science and technology projects, community involvement, and career preparation over four years of secondary education.

**Student:** Sarveshwar Senthil Kumar  
**Program:** SciTech (Science & Technology)  
**School:** Port Credit Secondary School  

## ✨ Features

- **Dark/Light Mode Toggle** - Switch between light and dark themes for comfortable viewing
- **Year-Based Navigation** - Browse portfolio by grade level (9, 10, 11, 12)
- **Organized Sections** - Each grade divided into clear categories:
  - Personal Information & Academic Achievement
  - SciTech Skills, Knowledge & Certifications
  - Community Work & Extracurricular Involvement
  - Advanced SciTech Experiences
- **Permanent Evidence Links** - All projects, reflections, and documents linked to Google Drive, GitHub, and external platforms
- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Social Media Integration** - Quick access to LinkedIn, GitHub, Devpost, and more
- **QR Code Access** - Easy mobile access to portfolio via QR code

## 🚀 How to Use

### Viewing the Portfolio

1. **Online:** Visit the live portfolio website
2. **Locally:** Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge)

### Navigation

- Use the **year tabs** at the top to switch between Grade 9, 10, 11, and 12
- Click on any **link** to view evidence, documents, or project artifacts
- Use the **dark mode toggle** in the header to switch themes
- Access **social media profiles** and **Drive folder** from the social links bar

## 🔧 Customization

To customize this portfolio for your own use:

1. **Edit Student Information:**
   - Update the header section in `index.html` with your name and details

2. **Update Links:**
   - Locate the `PERMANENT_LINKS` object in the JavaScript section
   - Replace URLs with your own Google Drive, GitHub, or other platform links
   - Update labels to match your projects

3. **Modify Requirements:**
   - Edit the `requirementsStructure` object to match your specific program requirements
   - Add or remove categories as needed

4. **Update Social Links:**
   - Modify the `SOCIAL_LINKS` array with your social media profiles

5. **Change QR Code:**
   - Replace the QR code image URL with your own portfolio URL
